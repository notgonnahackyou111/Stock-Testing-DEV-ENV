Bot API Documentation

Base URL

ws://localhost:8080 http://localhost:8080/api

Authentication

Include your Bot ID in all requests:

headers: { 'X-Bot-ID': 'your_bot_id_here', 'Content-Type': 'application/json' }

Available Endpoints

1. Place Order

POST /api/orders/place

{ "symbol": "AAPL", "side": "BUY", "quantity": 10, "orderType": "MARKET", "price": null }

2. Get Market Data

GET /api/market/{symbol}

Returns: price, bid, ask, 52W high/low, volume, market cap

3. Get Portfolio

GET /api/portfolio

Returns: cash, holdings, total value, P&L

4. Get Bot Stats

GET /api/bot/stats

Returns: trades executed, orders placed/executed/rejected, uptime

5. Subscribe to Updates

POST /api/subscribe

{ "subscriptions": [ "priceUpdates", "orderFills", "portfolioUpdates" ] }

WebSocket Events

  • price-update - Real-time stock price changes
  • order-filled - Order execution confirmation
  • order-rejected - Order rejection notification
  • portfolio-update - Portfolio value changes
  • market-data - Batch market data updates

Response Format

{ "success": true, "data": { ... }, "error": null, "timestamp": "2026-02-12T10:30:00Z" }

Code Examples

Python Bot Example

import requests import json BOT_ID = "your_bot_id" API_URL = "http://localhost:8080/api" headers = { 'X-Bot-ID': BOT_ID, 'Content-Type': 'application/json' } # Place a buy order order = { "symbol": "AAPL", "side": "BUY", "quantity": 10, "orderType": "MARKET" } response = requests.post( f"{API_URL}/orders/place", headers=headers, json=order ) result = response.json() print(f"Order placed: {result}") # Get market data market_data = requests.get( f"{API_URL}/market/AAPL", headers=headers ).json() print(f"AAPL Price: ${market_data['price']}") # Get portfolio portfolio = requests.get( f"{API_URL}/portfolio", headers=headers ).json() print(f"Total Value: ${portfolio['totalValue']}")

JavaScript/Node.js Bot Example

const axios = require('axios'); const BOT_ID = 'your_bot_id'; const API_URL = 'http://localhost:8080/api'; const headers = { 'X-Bot-ID': BOT_ID, 'Content-Type': 'application/json' }; // Place an order async function placeOrder(symbol, side, quantity) { const order = { symbol, side, quantity, orderType: 'MARKET' }; try { const response = await axios.post( `${API_URL}/orders/place`, order, { headers } ); console.log('Order:', response.data); } catch (error) { console.error('Error:', error.response.data); } } // Get market data async function getMarketData(symbol) { const response = await axios.get( `${API_URL}/market/${symbol}`, { headers } ); return response.data; } // Connect WebSocket const WebSocket = require('ws'); const ws = new WebSocket('ws://localhost:8080'); ws.on('open', () => { ws.send(JSON.stringify({ type: 'subscribe', botId: BOT_ID, subscriptions: ['priceUpdates', 'orderFills'] })); }); ws.on('message', (data) => { const event = JSON.parse(data); if (event.type === 'price-update') { console.log(`${event.symbol}: $${event.price}`); } if (event.type === 'order-filled') { console.log(`Order filled: ${event.orderId}`); placeOrder('GOOGL', 'BUY', 5); } });
