What is Stock Testing?

Stock Testing is a comprehensive platform for safely testing and validating trading strategies and automated bots without risking real money. Practice your trading skills, backtest strategies, and integrate your own trading bots in a realistic market simulation environment.

Quick Start Steps

Step 1: Start a Simulation

Click "Start the Test" on the home page. Configure your starting capital, risk tolerance, game mode, and difficulty level in the Options screen, then begin trading.

Step 2: Learn the Interface

Browse available stocks, view real price charts with technical indicators (20/50-day moving averages), and analyze stock statistics including 52-week highs/lows and trading volume.

Step 3: Execute Trades

Buy and sell stocks to build your portfolio. Monitor your profit/loss in real-time and adjust your strategy based on market conditions.

Step 4: Connect a Bot (Optional)

When ready, click "Hook up Bot" to connect your automated trading bot. Test it against live simulated market data.

Connecting Your Trading Bot

Stock Testing provides a complete REST API and WebSocket interface for connecting your trading bots:

Step 1: Access Bot Connection

Click "Hook up Bot" from the home page to access the bot management interface.

Step 2: Register Your Bot

Fill in bot details: name, type (Python, Node.js, etc.), API key for authentication, and optional description.

Step 3: View API Documentation

Check the API Documentation tab for complete endpoint specifications, request/response formats, and authentication details.

Step 4: Implement Bot Logic

Use the provided code examples as templates. Choose your preferred language (Python, Node.js) and integrate with the API endpoints.

Step 5: Start Trading

Launch your bot. It will communicate with the simulator in real-time, placing orders and receiving market data updates.

API Endpoints

Endpoint Method Purpose

Code Example - Node.js

const axios = require('axios'); const API_KEY = "your-bot-api-key"; const BASE_URL = "http://localhost:8000"; // Register Bot async function registerBot() { const response = await axios.post(`${BASE_URL}/api/bot/register`, { name: "MyNodeBot", type: "Node.js", api_key: API_KEY }); return response.data.bot_id; } // Get Market Data async function getMarketData() { const response = await axios.get(`${BASE_URL}/api/market/data`); return response.data; } // Place Order async function placeOrder(botId, symbol, action, quantity) { const response = await axios.post(`${BASE_URL}/api/bot/order`, { bot_id: botId, symbol: symbol, action: action, quantity: quantity, api_key: API_KEY }); return response.data; } // Get Portfolio async function getPortfolio(botId) { const response = await axios.get( `${BASE_URL}/api/portfolio?bot_id=${botId}` ); return response.data; }

‚ùì Frequently Asked Questions

What is the difference between Game Modes? +
Portfolio Building: Create a balanced portfolio with target allocations. You can rebalance anytime.
Day Trader: Limited daily trades (usually 3-5). Maximize daily profits.
Challenge Mode: Reach a 5% profit target before time runs out.
Can I connect a real trading bot? +
Yes! The platform provides a complete API for connecting trading bots written in any language. You can test your bot against simulated market data before deploying to real markets. Use the Bot Connection interface to register and manage your bots.
How is the market data generated? +
Market prices are simulated using realistic random walk algorithms that incorporate volatility, momentum, and trend factors. This creates realistic price movements while allowing you to test strategies without financial risk. You can also connect real market data APIs for authenticated price feeds.
What happens to my portfolio when I leave? +
Your portfolio data is stored in browser session storage. You can pause the market, close the browser, and return later to continue your session. For bots, connection data is stored in local storage and persists across sessions.
Can I trade on margin or short sell? +
The current version supports long positions only (no margin or shorting). These features are planned for future releases. Focus on building solid long strategies first!
Are there leaderboards or competitions? +
Leaderboards are coming in a future release. For now, track your own performance and compare results with friends by sharing your stats. Bot performance dashboards provide detailed metrics on strategy effectiveness.
How accurate is the simulation? +
The simulator uses realistic algorithms for price movements, including volatility, momentum, and trend factors similar to real markets. However, it's a simplified simulation. Factors like slippage, commissions, and real market liquidity are not included. Use it as a testing tool, not perfect backtesting software.
Can I use this to learn trading? +
Absolutely! Stock Testing is designed for education and strategy testing. Start with Easy mode, learn the interface, build some strategies, then graduate to harder modes. It's a safe, risk-free way to learn trading without losing money.

üîß Troubleshooting

Chart not loading?

  • Refresh the page
  • Clear browser cache (Ctrl+Shift+Delete)
  • Make sure you've selected a stock

Bot not connecting?

  • Verify your API key is correct
  • Check that the bot name and type are filled in
  • Ensure you're on the correct page (bot-connection.html)
  • Check browser console for error messages (F12)

Prices not updating?

  • Click the Play button to resume market simulation
  • Check the Time Speed selector (should not be 0x)
  • Try refreshing the page

Portfolio calculations look wrong?

  • Verify your buy prices against current price
  • Remember that display updates in real-time
  • Check the transaction history

Session keeps resetting?

  • Don't clear browser data while trading
  • Use the same browser to continue sessions
  • Enable storage in browser settings
